Угадай мелодию. Выберите любимого исполнителя и проверьте себя на знание его композиций. Вам предлагается тест из 10 музыкальных вопросов с 5 вариантами названий композиций, в каждом вопросе с произвольного момента включаются песни. Программа сама скачает нужные композиции.


*****
Для работы программы необходимы интерпретатор python и библиотеки для python:
requests
bs4
lxml

python можно установить с сайта разработчика бесплатно. В ходе установки убедитесь, что опция "Add Python to PATH" была активирована.
Для установки библиотек необходимо в cmd ввести команды:
python -m pip install requests
python -m pip install bs4
python -m pip install lxml

Также необходимо, чтобы в папке с игрой (файл с расширением exe) находился файлы NAudio.dll и music_downloader.py
